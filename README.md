# Hexagon Conquest

## Game Documentation

Hexagon Conquest is an engaging strategy game where players compete to control territory on a hexagonal grid. Players must outmaneuver their opponents while managing their resources and making strategic decisions to dominate the game board.

### Gameplay Overview
- Players take turns to conquer new hexagons and expand their territories.
- Strategic resource management and tactical decision-making are key to winning.

### Features
- Unique hexagonal grid gameplay.
- Customizable game settings.
- Various difficulty levels for both beginners and experienced players.

## Installation Instructions

To install Hexagon Conquest, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Phu2812/hexagon-conquest.git
   ```
2. Navigate to the project directory:
   ```bash
   cd hexagon-conquest
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the game:
   ```bash
   npm start
   ```

Enjoy conquering hexagons!
